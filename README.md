# RegenX

RegenX is a game engine made with two goals in mind.

## Goals

### 1. Easy multi-language development

RegenX aims to support multiple programming languages with the same level of control.

Planned supported languages:
- C++
- C#
- Python
- Lua

Language support means the ability to **compile, load, and execute scripts**, not just having engine code written in that language.

All supported languages should be able to interact with the engine in a comparable way.

---

### 2. Retro console support

RegenX is planned to support retro and older consoles, including:
- PlayStation 1
- PlayStation 2
- PlayStation 3
- Xbox 360
- Nintendo 3DS
- Nintendo Wii
- Nintendo GameCube
- and more

This includes adapting the engine to the limitations and toolchains of each platform.

---

## Current Focus

RegenX is currently focused only on **goal 1**.

Work on retro console support will not begin until the engine matures and the core systems are stable.

## License and Usage

The RegenX engine itself is open source.

- Forking or modifying the engine must remain open source.
- Closed-source forks of the engine are not allowed.

Games and applications built using RegenX may be closed source, as long as the engine itself is not modified and kept private.

Proper credit is required when using RegenX.

See the `LICENSE` file for full details.

---

## Project State

RegenX is under active development.

APIs may change.
Modules may be refactored.
Expect breaking changes as the engine evolves.
